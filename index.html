Stone Sky Sea DPS Calculator for FFXIV

To use the Stone Sky Sea Calculator you will need to note down the time remaining if you didn't kill the dummy or the HP remaining (in %) if you killed him.

If you can't see the HP remaining go to System > Character Configuration > UI Settings and make sure "Display target's remaining HP" is turned on.
